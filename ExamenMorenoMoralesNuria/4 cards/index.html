Ejercicio

Trata de recrear la landing page tal y como aparece en las imágenes de abajo.

Reglas:

  • No es obligatorio usar el HTML y CSS que se proporcionan. Si el alumno lo desea puede empezar el ejercicio desde cero.
  • Tanto el HTML como el CSS que se proporcionan pueden ser modificados.
  • Se puede usar cualquier tipo de letra sin serifa.
  • El ancho máximo que ocupe el contenido no debe superar los 1115px en escritorio (centra el contenido si la pantalla es mayor) y 375 en móvil.
  • Cada tarjeta tiene un ancho fijo de 350px y un alto de 250px.
  • El cambio de layout se produce a los 1200px.
  • Es obligatorio usar variables CSS para los colores que son estos.
    • Rojo: #e85454
    • Cian: #45d1d0
    • Naranja: #f9ad4c
    • Azul: #539ced
    • Fondo: #fafafa
    • Color de fondo de las tarjetas: #fff
    • Color del texto: #456;
  • No se necesita (y no se debe usar) Javascript.
